# Industrial Dryer Monitoring System

A dashboard for monitoring and managing industrial dryer operations. Track temperature, humidity, and air flow metrics, manage drying sessions, and view historical reports.
This project is just a prototype.

## Features

- Real-time sensor monitoring (temperature, humidity, air flow)
- Session management with status tracking
- Historical reports and analytics
- Responsive design for desktop and mobile
- Dark/light theme support
- Secure authentication

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Radix UI components(Shadcn/ui)
- React Router
- Recharts
